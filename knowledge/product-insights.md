# 제품 기획 인사이트

## 동물상 앱 (Apex Aura) 핵심 가치
**재미 (동물상 + 뛰어난 UI) + 희망 (개선 지침)**
- 재미 없으면 공유 안 됨 → 바이럴 없음
- 희망 없으면 Pro 전환 없음 → 수익 없음
- 모든 화면 설계 이 두 축으로 평가할 것

## 페이월 설계 원칙
- Free가 너무 많이 주면 Pro 전환 동기 없어짐
- Free: 첫 번째 액션카드만 공개, 나머지 잠금 티저
- 컨설턴트 리포트: Free는 quote 1줄만, Pro는 5요소 풀버전
- 공유 가능한 화면이 바이럴의 핵심 (공유 전용 카드 필요)

## 콘텐츠 벤치마크
- 16Personalities: 결과 스크린샷이 그 자체로 콘텐츠 → 공유 카드 설계 필수
- 퍼스널 컬러: 구체적 색상 코드 + 쇼핑 연결 → 어필리에이트 구조
- Looksmaxxing: 수치 기반 + 구체적 액션 → ML Kit 수치 활용

## 납득감 설계
- "왜 이 동물상인가" 설명 없으면 이탈률 높음
- ML Kit 수치와 동물상 기준을 비교해서 보여줄 것
- "눈꼬리 -2.3°라서 강아지상" 한 줄이 결과 화면에 보여야 함

## AI 콘텐츠 퀄리티
- "일반론 패턴" 차단 필요: "레이어드컷은 윤곽을 강조해요" 같은 교과서 표현
- 메커니즘 4요소 강제: 관찰 → 영향 → 변화 → 결과
- 셀럽 레퍼런스 3요소: 장면(작품+연도) + 연결이유 + 포인트
