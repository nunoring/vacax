## Chapter 1.1 — 인간의 중독과 공포 (마약상과 살인자) (p10~16) <a id="chapter-1"></a>

### 마약상 6단계 판매 공식 → "마약" 자리에 "지식"으로 치환
1. 황홀한 경험을 보여준다 (체험·맛보기)
2. 무료/헐값 제공
3. 자발적 요구할 때까지 중독시킨다 (콘텐츠 누적)
4. 고객이 "제발 팔아달라"는 단계까지 끌어올린다
5. 초고가 판매
6. 1~5 반복

지식 상품의 이점: **합법** + **무형이라 무한복제·확장 가능**.

### 살인자에게서 배운 2가지
- **인간의 충동성**: "갖고 싶다"가 강하게 들면 가격·리스크 무관하게 구매. 좋은 상품이라도 욕망 없으면 안 팔림.
- **대중의 과몰입**: 충격적 자극에 사람들이 자기 시간 희생하면서까지 몰입한다 (조두순·유영철 예시). 일단 몰입시키면 자기 것 다 잊는다.

### 결론
- 마약 팔듯 사업하되 마약은 팔지 마라
- 고객을 몰입시켜라
- "미칠 듯이 갖고 싶다" 만들어라
- 누구든 배움의 자세 (대상 무관)

→ **VacAX 적용**:
- **무형 지식/콘텐츠/SaaS 방향성 OK** (VoiceFit·apex_aura가 이 방향).
- 제품 마케팅 = "이런 가치가 있습니다" X → "당신이 미칠 듯이 갖고 싶어할 이유" 직접 자극.
- 무료 체험·맛보기 단계 의도적 설계 필수 (VoiceFit 무료 1회 체험 → 유료 전환 등).
- 윤리 경계: 자극·도파민 자극을 자율 위협 수준까지 끌지 말 것. **knowledge/product-insights.md에 "후킹 전 가치 검증" 안전장치 추가 검토.**

## Chapter 1.2 — 인간 불변의 습성 6가지 (p17~42) ★★★

핵심 명제: "인간은 늘 **생존에 유리해 보이는** 선택을 한다 (실제 유리한 게 아님)"

### 6가지 습성

**1. 필요착각** — 필요한 게 아니라 "갖고 싶은 것"을 산다
- 사람을 갖고 싶게 만들려면:
  - 돈/건강/관계에 직결된 상품 (본능 시스템 1차 문지기 즉각 반응)
  - "**실망 위기**"를 안겨주기 — 고객은 실망 피하려고 큰 비용도 지불

**2. 자기무결** — 자신의 판단을 부정 못함
- "확실한 이득 > 불확실한 손실 > 확실한 손실" (대니얼 카너먼 전망 이론)
- 적용 예시:
  - 사업가가 푼돈에 만족하는 이유: 큰 사업 = 불확실한 이득, 작은 이득 = 확실
  - 직장인이 못 벗어나는 이유: 월 1억 도전 = 불확실 / 월급 300 = 확실
  - 도박/주식 손절 못함: 손절 = 확실한 손실 / 버티기 = 불확실한 손실
  - **구매 단계가 어려울수록 잘 팔림**: 이미 시간·노력 투자한 상황에서 "안 사면 그 비용 다 날리는 확실한 손실"이 됨

**3. 집단의존** — 개인은 집단을 이길 수 없다
- 개인 1:1 = 각자 다름. **집단은 획일적, 정해진 공식 통함**
- 적용:
  - 1:1 컨설팅·맞춤 서비스 = 어렵고 수익성 낮음 → 본수익 X
  - **다수 동시 판매 가능 상품 (구독, PDF, 강의, SaaS) = 본수익 O**
  - "22살 김민수"는 다르지만 "20대 남자"는 획일적 — 개인 차이 무시, **집단의 평균 성향에 초점**
  - 내가 원하는 특성의 개인들을 미리 모아 집단 형성 → 신입은 자동 동화
  - 고객들 간 **계급 두기** (등급별 혜택 명시)

**4. 독점위선** — 모두가 성공하는 세상 원하지 않음
- 인간은 자원 독점 욕망 + 이타적 가면. 모순적·이기적.
- 적용: **고객의 평판을 지켜주면서 독점적 이득 제공**. 예: "이건 비밀로 해주세요" 특혜.

**5. 효율우선** — 에너지 소모 최소화 본능
- 인간은 사고 활동을 본능적으로 피함. "절전 모드" 선호.
- 적용:
  - **추상적 설명으로 시작 X**. 직관적 후킹(본능 시스템)으로 먼저 → 사고 시스템 가동
  - **자유로운 선택 싫어함**. 백지보다 제한된 선택지 선호 → 의도적으로 선택지 제한
  - "고민하기 귀찮아서" 충동적으로 고가 결제하는 경우 많음
  - 지식 가격은 단순히 욕망 크기의 지표 (절대값 없음)

**6. 무상탈취** — 공짜만 원함
- "호의로는 호의를 살 수 없다" — 치알디니 상호성 법칙 반박
  - 상호성 법칙은 **나와 상대가 가치 수준이 비슷할 때만** 작동
  - 가치 낮은 상대에겐 시늉만, 진짜 호의는 가치 높은 사람에게만
- **고객에게 잘해주려고만 하지 마라.** 잘해줄수록 "하찮은" 취급 받음
- "부탁은 거절당한다" — 부탁 = 가치 낮아 보임 → 갑질 욕구 유발
- 적용:
  - 고객에게 **임무를 던지고 보상·손실 암시** (부탁 X)
  - 공짜 미끼는 OK, but 본 상품 가치 메시지 분명히
  - 자신을 "가치 높은 사람"으로 포지셔닝 우선

→ **VacAX 적용** (이 챕터가 가장 중요):
1. **개인 1:1 컨설팅·맞춤 서비스는 본수익 X** (현재 voicefit·apex_aura가 SaaS·제품 방향인 건 OK)
2. **포트폴리오·이력서 톤**: "잘 해드리겠습니다" X → "이런 결과를 만들 수 있는 사람" (가치 높은 사람 포지셔닝). 부탁/저자세 X.
3. **CLAUDE.md 반영 후보**: "고객에게 잘해줄수록 가치 낮게 인식된다 — 무한 무료·덤 정책 금지. 가치 메시지 명확히 + 적임자 선별"
4. **카피라이팅 룰**: 실망 위기 자극 + 본능 시스템 먼저 자극 + 선택지 제한
5. **윤리 경계**: 책 톤은 "악용"에 가까움 — 자극 강도 조절. VacAX는 "정직 + 후킹"의 균형 유지

## Chapter 1.3 — 절세 편법 (p43~49) ★★★ 실무 직결

### 공식: 면세법인 + 청년창업중소기업 세액감면 + 비상주 오피스

## Chapter 2.1 — 무의식 카피라이팅 (p50~73) ★★★★ 최고 응용도

### 시대적 명제
- 인간 사고 시스템이 점점 퇴화 (책 → 신문 → 블로그 → 페북/인스타 → 유튜브 → 틱톡)
- 이제 **3초 안에 한 문장으로 후킹 못 하면 망한다**
- 즉흥적 보상 체계에 빠진 대중 = **유혹당하기 쉬운 환경** (사업 기회)

### 6가지 상자 (카피라이팅 통로)

| 상자 | 핵심 | 어휘 예시 |
|---|---|---|
| **1. 욕구성** | 돈/건강/관계와 연관 짓기 | "돈 버는", "취업되는", "건강 유지하는" |
| **2. 날먹성** | 날로 먹는 기분 (효율우선+무상탈취) | "쉽고 간단한", "그대로 따라하는", "한번 알면 평생" |
| **3. 충격성** | 몰입도를 계단식으로 올림 | 의외성/확신성/도발성/위험성 |
| **4. 희소성** | 귀하게 만들기 | 비밀성/고난성/한정성/참신성 |
| **5. 특정성** | 생각의 양 줄여주기 | 구체 수치, "이것", "공식/법칙/루틴" |
| **6. 정체불명** | 낯선 명칭 (기억·회자 유도) | 새로운 작명. 단독으론 X — 매력 문구와 동반 필수 |

### 통과 예시 진화 (글쓰기 사업 카피)

```
원문: 글쓰기
+욕구성: 글쓰기로 돈 버는 법
+날먹성: 글 몇 줄 그대로 따라 써서 돈 버는 법
+충격성: 머리 나쁠수록 더 유리한 글 몇 줄로 경제적 자유 얻는 법
+희소성: 머리 좋으면 절대 모르는 글 몇 줄로 경제적 자유 얻는 비밀
+특정성: 머리 나쁜데 월 1억 버는 사람들만 몰래 쓰는 글쓰기 루틴 2가지
+정체불명: 머리 나쁜데 월 1억 버는 사람들만 몰래 쓰는 글쓰기 기술 '조립 루틴'
```

### 정체불명 상자의 추가 효과
독자적인 명칭(이론·기술·메서드명)을 누적할수록 **세상 유일무이한 정체성** 구축 → **가격 결정권** 확보 (가격경쟁 회피 가능).

### 운용 원칙
- 6개 다 통과 안 해도 OK. 적절한 조합 추출이 핵심.
- 무조건 많이 거치면 이전 효과가 사라지기도 함.

→ **VacAX 적용** ★★★★:
1. **포트폴리오 제목·이력서 한 줄 요약·블로그 제목·영상 썸네일·LinkedIn 헤드라인** 모두 6상자 체크 적용

## Chapter 2.2 — 뇌과학 글공식 6단계 (p74~103) ★★★★★ 가장 실용

### 6단계 템플릿
**모든 글쓰기·말하기·콘텐츠**에 적용 (영상 대본, 블로그, 강의, 상품 페이지, 컨설팅, 포트폴리오 등).

| 단계 | 목적 | 분량 | 중요도 |
|---|---|---|---|
| 1. **메인 후킹** | "어, 이게 뭐지?" | 5% | ★★★★★ (가장 중요) |
| 2. **서브 후킹** | "왜 계속 봐야 하지?" | 중간 | ★★★★ |
| 3. **효과 입증** | "와 진짜 되네!" | 중간 | ★★★★ |
| 4. **전문가 포지셔닝** | "오 진짜 전문가네!" | 90% (본론) | ★★ (가장 낮음) |
| 5. **콜투액션** | "이것만 하면 된다는 거지?" | 짧음 | ★★★★ |
| 6. **서브 후킹 (마무리)** | "정말 무조건 해야겠는걸!" | 짧음 | ★★★★★ (정점·종점 원칙) |

### 1. 메인 후킹 (5%, but 사업 성패 결정)
- 썸네일, 제목, 첫 장면/멘트, 첫 줄, 슬로건, 브랜드명
- **무의식 카피라이팅 6상자**로 생산

### 2. 서브 후킹 (집중 유지)
- **얻을 이득 먼저 명시** (집중 이어갈 이유)
- **엥커링** (소품·BGM·말투·캐릭터로 감정 형성 → 기억에 새김)
- 인간은 **언어가 아닌 감정**으로 기억
- "이득을 주고 기억에 남기면 끝난다" — 부정적 감정이라도 강렬하면 OK

### 3. 효과 입증 (가장 직관적인 증거)
- "이게 진짜 효과 있구나" 확신 주는 단계
- 장황한 설명 X, **직관적 시각 자료 O** (방탄유리에 AK 총알 갈기듯)
- **본인 성과 없을 때 → 외부 성과 레버리지**:
  - 뉴스 보도 사례, 유명인 사례, 지인 사례, 차트
  - **단 조건**: 그 원인·기술을 분석해서 사람들에게 가치 공유 가능해야 함 (아니면 사기꾼)
  - 예: 부동산 투자 성과 없이도 외부 시세차익 사례 분석으로 사업 가능

### 4. 전문가 포지셔닝 (본론, 분량 가장 많지만 중요도 가장 낮음)
- 대부분 사람들이 본론이 핵심이라 착각 → 망함
- 본론의 유일한 목적: **"나는 전문가다"** 메시지 전달
- 99%는 본론 내용 어차피 잊음
- 본론을 정성껏 채워야 하는 3가지 이유:
  1. 효과 입증이 운이 아니라 인과관계라는 신뢰 제공
  2. 실행하는 1% 극소수가 사업을 견인
  3. 전문가는 콘텐츠·상품 가격이 높아짐
- 실행: 해외 연구 결과 인용, 학술 이론, 전문가 명언 등

### 5. 콜투액션 (CTA) ★ 핵심 룰
- 다음 액션 명시 (다음 영상, 댓글, 이메일, 커뮤니티, 독서 등)
- **주의 1**: 상업적 티 X (결제 페이지 직전까지는 모든 게 무료처럼 느끼게)
- **주의 2**: **제공자 태도** (지배형) vs **요청자 태도** (종속형)

| 종속형 (요청자) | 지배형 (제공자) |
|---|---|
| "꼭 읽어주시길 부탁드립니다" | "읽으셔야 불이익 없습니다" |
| "구매 방법은 아래와 같습니다" | "구매 방법은 아무나 찾을 수 없게 숨겨두었습니다" |
| "언제든 누구든 들어오실 수 있습니다" | "인원이 많아 조절 중입니다" |
| "포기 안 하시게 도와드리겠습니다" | "진입장벽은 점점 높아지니 자신 없으면 포기하세요" |
| "후회 없으실 거예요" | "망설여지면 제발 사지 말아주세요. 다른 적임자에게 양보를." |

- 핵심 패러다임: "어디, 네가 평범한 멍청이들과 다른지 한번 볼까?" / "딱히 필요하진 않은데, 원하면 도전해봐"

### 6. 서브 후킹 마무리 (정점·종점 원칙)
- 카너먼: 인간은 마지막 감정을 가장 오래 기억
- 핵심: "내가 알려준 액션 취하면 손쉽게 공짜로 다 가짐. 안 하면 엄청난 손해" 느낌
- 카피라이팅 6상자로 생산

→ **VacAX 적용** ★★★★★ (가장 강력한 챕터):

1. **knowledge/copywriting-rules.md 신규 작성** (필수):
   - 6상자 카피라이팅 (제목·헤드)
   - 6단계 글공식 (전체 콘텐츠 구조)
   - 제공자 태도 vs 요청자 태도 체크리스트

2. **포트폴리오/이력서 재구조화**:
   - **메인 후킹**: 한 줄 헤드 ("[N년 안에] [어떤 결과를] [어떻게] 만든 [정체불명 메소드]")
   - **서브 후킹**: 채용 담당자가 얻을 이득 (예: "이 사람을 채용하면 어떤 문제를 어떻게 풀 수 있는지")
   - **효과 입증**: 정량 성과 시각화 + 외부 레퍼런스 (정량 약하면 외부 성과 레버리지로 보강)
   - **전문가 포지셔닝**: 학습/도구/방법론 (분량 많아도 중요도 낮음 인지)

   - **서브 후킹 마무리**: 인상 깊은 한 줄

## Chapter 3.4 — 공짜 곡괭이 이론 (p137~170) ★★★★ 무료/유료 구분

### 핵심 명제

### 무료/유료 구분 기준
| 무료 (공짜 곡괭이) | 유료 |
|---|---|
| 누구나 쉽게 이해, 즉시 사용 가능 | 이해·사용이 어려운 것 |
| 자극적, 동기부여에 도움 | 소수만 알아야 가치 유지되는 것 |
| 효과가 빨리 나타나는 것 | 효과가 늦게 나타나는 것 |

### 공짜 곡괭이 4가지 역할

**(1) 진입 유도** — 모든 게 무료처럼 보이기
- 상업적 티 절대 X (유료 상품 존재 노출 X)
- "강의/교재/코칭/구매 방법" 같은 메뉴 X → "신청하는 방법", "얻는 방법" (정체불명 상자)
- 유료 결제 직전까지 무료/유료 헷갈리게
- 흔한 실수: "여기까진 무료, 더 보려면 강의 결제" → 미친 짓. 매출 폭락.
- 대안: "다 알려주고 싶지만 시간상/외부 요인으로 다른 곳에 정리해 두었다"

**(2) 투자 유도** — 항상 할 것 주기 (숙제)
- 추상적 지출(시간·노력) 유도. 돈 X (사업 초반)
- 인간 뇌는 돈에는 민감, 시간·노력엔 둔감
- 예: 미라클 모닝 — 책 내용 허접하지만 "할 것"을 줘서 베스트셀러
- 조작연애 TV — 소개팅 앱 시도, 칼럼 정주행, 책 26권 읽기, 글 작성 등 끊임없이 숙제 제공
- 콘텐츠 발행 주기 길어도 OK. 단 숙제를 줘서 고객 잡아두기

**(3) 고객 테스트** — 끊임없이 테스트, 진짜 피드백 수집
- **설문조사 = 쓸모 없음**. 기계적 답변만.
- 진짜 피드백: 실제 사용·문제 발생 관찰 + 건의사항 경청
- 메타인지 향상 (= "내가 모르는 걸 아는 능력") — 경제적 자유의 초능력
- 사업 초반 돈 받기 시작하면 제약 폭증 → 일단 무료로 검증 우선

**(4) 성과 축적** — 환호성 만들기
- 카지노가 일부 잭팟 터뜨리는 원리
- 실제 성과를 거둔 소수의 후기·환호성 공개 노출
- 효과: 안 된 사람도 "내가 문제구나, 더 노력하자" 또는 "더 좋은 게 있나? 돈 내서라도"
- 공짜로 잘 된 사람일수록 유료 상품도 살 확률↑ (인간은 99%→100% 만드는 데 거액 지불)

"내가 고객에게 공짜 곡괭이를 준다 = 고객에게서 무료로 받는 게 훨씬 크다"
