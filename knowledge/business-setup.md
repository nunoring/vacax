# 사업자 등록 전 체크리스트 (절세 편법)

출처: `경제적 자유 초월차선` Ch1.3 절세 편법
적용 시점: **첫 매출 발생 직전 / 사업자 등록 직전 (한 번뿐인 기회)**

⚠️ **놓치면 영구 손실**: 청년창업중소기업 세액감면은 **생애 첫 창업에만 적용**됨.

---

## 핵심 공식

**면세법인 + 청년창업중소기업 세액감면 + 비상주 오피스 = 연 세금 거의 0원**

1. **법인사업자** (개인사업자 X — 연 1.5억 이상 예상 시)
2. **면세사업자** 등록 (지식창업이면 대부분 가능 → 부가세 10% 면제)
3. **청년창업중소기업 세액감면** (만 15~34세, 생애 첫 창업만)
   - 수도권과밀억제권역 **외** = 5년간 법인세 **100% 감면**
   - 수도권과밀억제권역 **내** = 50% 감면
4. **비상주 오피스** 임대 (수도권 외 지역) + 정기적 법인카드 사용 (세무조사 대비)

---

## CEO 현 상황 적용

- 임용후 (1990년대 중후반 출생 추정) → **만 34세 이하면 청년창업 감면 대상**
- 1순위가 AX/AI 기획 이직 포트폴리오긴 하나, 사이드로 사업자 등록 검토 시점이 오면 이 공식 반드시 적용
- VoiceFit/apex_aura 등 수익 발생 직전이 결정 시점

## 절차 (책 인용 + VacAX 조정)

```
(1) 사업 시스템을 수익화 직전까지 완성 (사업자 등록 X)
(2) 능력 있는 세무사 방문 상담 요청
(3) 면세사업자 등록 가능 업종으로 정해 달라고 요청
   - 무형 상품(SaaS·콘텐츠·앱·교육 등) = 대부분 가능
(4) 세무조사 비교적 안전한 수도권과밀억제권역 외 지역 추천 요청
(5) 추천 지역에 비상주 오피스 임대
(6) 면세법인 사업자 등록 (개인사업자 X)
(7) 5년 후 청년창업중소기업 세액감면 종료 시점:
   - 가족·신뢰 지인 중 청년(만 15~34) 명의로 다시 등록
   - 단 대표자 잘못 세우면 법인 통째 빼앗길 위험 — 신뢰 검증 필수
(8) (1)~(7) 반복
```

---

## 비상주 오피스 사용 시 주의

- 사업장 주소만 그 지역, 실제 활동은 다른 곳 → 세무조사 시 의심
- **그 지역에서 정기적 법인카드 사용 필수**:
  - 방법 1: 매주 직접 방문해서 그 지역 식당·카페에서 법인카드 사용
  - 방법 2: 그 지역 거주 지인에게 법인카드 한도 정해 발급 → 의무적 사용 지시 (지인 입이 찢어짐)
- 번거로우면 차라리 **집 주변 사업장 + 50% 감면**으로

---

## 사전 확인 사항 (세무사 미팅 전)

- [ ] 본인 첫 창업 여부 (이전 사업자 등록 이력 0건이어야 함)
- [ ] 만 34세 이하 여부 (창업 당시 기준)
- [ ] 사업 아이템이 면세 업종 해당 가능한지 (지식창업이면 대부분 OK)
- [ ] 수도권과밀억제권역 외 비상주 오피스 후보 지역 리스트업
- [ ] 5년 후 명의 이어받을 신뢰 청년 후보 (선택)

## 세무사 미팅 시 질문 리스트

1. "제 사업이 면세사업자 등록 가능한 업종으로 정해질 수 있나요?"
2. "청년창업중소기업 세액감면 받을 수 있는 조건이 모두 충족되나요?"
3. "세무조사 시 비교적 안전한 수도권과밀억제권역 외 지역 추천 부탁드립니다."
4. "비상주 오피스 사용 시 법인카드 사용 패턴 주의사항이 있나요?"
5. "세법이 최근 1년 내 바뀐 부분이 있나요?" (세법은 수시 변경)

## 세무사 선택 원칙

- 세무사는 **나의 동업자/선생님 아님**. 내가 고용한 사람일 뿐.
- 세무사가 알아서 절세해 줄 것 명신 금지.
- **내가 미리 공부하고 요구할 줄 알아야** 세무사도 도움 줌.
- "이런 거 알고 계신가요?" 식으로 적극 질문.

## 절약 규모 (참고)

책에서 저자는 이 공식으로 "수십억 이상 절약" 주장. VacAX 규모 예상:
- 연 수익 1억 → 부가세 1000만 + 법인세 약 1500만 절약 = **연 2500만 절약**
- 연 수익 10억 → 절약 규모 **연 2~3억** 수준
- 5년 간 누적 = **수십억 단위 가능**

→ AX/AI 기획 이직 + 사이드 사업 동시 운영 시 사이드 사업 사업자 등록 시점에 반드시 적용.
