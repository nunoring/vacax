# HairFit MVP STATE

Updated: 2026-07-02

## Project

## Blocker
- 포트폴리오용 그래픽 polish 전. 앱 내부 UI 1차 리프레시는 창백해서 재디자인 필요.
- 실제 사람/현장 사진 기준 얼굴 유지 품질은 아직 미검증. 랜덤 인터넷 얼굴 대신 synthetic 5장 또는 동의/라이선스가 명확한 사진만 사용.
- 브라우저 자동화 표면에서 파일 선택 주입은 미지원. 업로드 포함 UI 검수는 실제 화면에서 수동 1회 필요.
- 모든 샘플이 완전 동일인은 아님. 긴 기장/리프 계열 reference 리스크는 낮췄지만 실제 고객 사진 기준 재검증은 필요.
